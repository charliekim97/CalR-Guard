# Changelog

All notable changes to this project should be documented here.

## [0.3.1] - 2026-03-22

### Added
- GitHub-ready repository scaffolding
- CI workflow for tests on push and pull request
- issue templates and pull request template
- contributing guide, security policy, code of conduct
- publish-to-GitHub instructions and public release checklist

### Changed
- removed placeholder author metadata from `pyproject.toml`
- added optional development dependency group for `pytest`

## [0.3.0] - 2026-03-22

### Added
- modular Python package structure
- local Gradio app and CLI
- CSV / TSV / TXT / XLSX import support
- metadata merge support
- suggested exclusions derived from QC flags
- review report and QC plots
- bundle export and smoke tests
